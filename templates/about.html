What is the Learning Guidance System
The learning guidance system is a platform where individuals can create syllabi that others can follow to learn new things.
Why Was the Learning Guidance System Created
The Learning Guidance System was created becuase school is expensive and libraries already provide free knowledge. The Learning Guidance System was created to provide a replacement for the advice provided in schools.
How You Can Use the Learning Guidance System
You can look up subjects and learn about them or create your own.
